Cette fenêtre contient du nouveau contenu. L'on peut s'en servir de différentes façons mais n'oubliez jamais de considérer la convivialité de votre application web lorsque vous ouvrez une nouvelle fenêtre. Dans plusieurs cas, il est préférable d'avoir une page avec du contenu dynamique que l'on met à jour avec le paradigme Ajax plutôt que d'ouvrir une nouvelle fenêtre. Par contre, dans certains cas, l'ouverture d'une nouvelle fenêtre est vraiment la fonctionnalité qu'il nous faut.
